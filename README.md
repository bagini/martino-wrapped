## Deploy to Vercel

### Option 1: One-Click Deploy (Easiest)

1. Push this folder to a new GitHub repo
2. Go to [vercel.com/new](https://vercel.com/new)
3. Import your repo
4. Click **Deploy**

### Option 2: Vercel CLI

```bash
npm i -g vercel
cd martino-wrapped
vercel
```

## Local Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)
